Для чего нужна программа SyncBooker?

Программы SyncBooker предназначена для синхронного просмотра и прослушивания электронных книг.

Сами книги в этой программе содержат метки на время воспроизведения и позиции в тексте, что позволяет удобно и легко выполнять скороллинг в нужное место.

Её можно использовать как обычный плеер, так и ридер книг.

Единственное ограничение программа - бинарник только для платформы Windows x64.

Все книги в каталоге есть на двух языках - русском и английском (добавление других языков не планируется, хотя Вы можете сделать это самостоятельно.)

Можно ли использовать программу SyncBooker для изучения языка?

Да, конечно. Многие для изучения иностранного языка используют иностранные фильмы с субтитрами.

Программа SyncBooker позволяет делать практически тоже самое, но с помощью электронных книг.

Какие планы в развитии программы SyncBooker?

Во-первых, постоянная работа над улучшением синхрона как каждого языка, так и между языками.

Во-вторых, конечно, пополнение каталога новыми книгами.

В-третьих, добавление в интерфейс возможности самостоятельного создания книг. Для этого потребуется версия программы для разработки, она занимает гораздо больше места, но при этом позволит создавать синхронные книги с помощью графического интерфеса программы, будет помогать улучшать качество синхрона. Замечу, что это достаточно сложная задача и отдалённое будущее.

В-четвёртых, для уменьшения распространяемого бинарного дистрибутива есть мысль добавить в программу возможность скачивания обновлений каталога через распределенную сеть DHT, но эта возможность тоже может потребовать как больших усилий и времени разработки, так и наличие желания пользователей запускать такую программу для раздачи книг другим пользователям.
